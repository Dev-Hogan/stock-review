## 目标

- 能够获取 A 股股票信息，历史信息，获取分钟级数据等
- 在 web 页面通过自然语言描述，大模型自动生成筛选因子代码
- 能够根据给定的模式或行为，筛选出符合条件的股票
- 提供可视化配置界面和 HTML 报告输出
- **Web 前端交易界面**:
  - 自选股池管理 (添加/删除股票)
  - 个股 K 线图表实时展示
  - 历史盘口回放功能 (复刻每日盘中情况)
  - **多股同步回放**: 支持同时查看多只股票的历史分时回放，时间轴同步
  - 右侧面板显示：逐笔成交、成交明细、买卖五档
  - 参考同花顺界面风格，简化设计

## 技术栈

- **后端**: Python 3.11+
- **前端**: TypeScript + Vue 3 + Vite
- **数据获取**: akshare
- **数据存储**: SQLite
- **大模型**: 本地部署 (通过 Ollama 调用)
- **图表库**: Lightweight Charts (TradingView)
- **依赖管理**: Poetry (后端) / pnpm (前端)
- **并发处理**: asyncio + aiohttp

## 功能模块

- [ ] 获取 A 股股票基本信息 (名称、代码、行业、市值等)

- [ ] 获取分钟级数据 (1 分钟/5 分钟/15 分钟/30 分钟/60 分钟)

## 非功能性需求

- **性能**: 
  - 全市场 5000+ 股票筛选应在 5 分钟内完成
  - 多股回放 (4 股以上) 流畅播放，无明显卡顿
  - 图表渲染响应时间 < 200ms
- **可靠性**: 数据获取失败自动重试 3 次
- **可扩展**: 易于添加新的因子模板和数据源
- **安全性**: 生成的代码需安全校验，禁止危险操作

## 数据库设计

### 股票基础信息表 (stocks)
- code: 股票代码
- name: 股票名称
- industry: 行业
- market_cap: 市值

### 日 K 线数据表 (daily_klines)
- stock_code: 股票代码
- date: 日期
- open/high/low/close: 开盘/最高/最低/收盘
- volume: 成交量
- amount: 成交额

### 分钟 K 线数据表 (minute_klines)
- stock_code: 股票代码
- datetime: 日期时间
- open/high/low/close/volume/amount

### 分时数据表 (time_shares)
- stock_code: 股票代码
- trade_date: 交易日期
- time: 时间 (秒级精度，如 09:30:15)
- price: 成交价格
- volume: 成交量

### 逐笔成交表 (trades)
- stock_code: 股票代码
- trade_date: 交易日期
- time: 成交时间 (精确到毫秒)
- price: 成交价格
- volume: 成交数量
- direction: 买卖方向

### 五档行情表 (order_books)
- stock_code: 股票代码
- trade_date: 交易日期
- time: 时间点 (秒级)
- bid1-5_price: 买一到买五价格
- bid1-5_volume: 买一到买五数量
- ask1-5_price: 卖一到卖五价格
- ask1-5_volume: 卖一到卖五数量

### 自选股表 (watchlists)
- id: 主键
- stock_code: 股票代码
- category: 分类
- sort_order: 排序
